this is my golang task manager using the cobra


## installation:
```bash
cd
git clone github.com/reposwagod/cliTaskManager.git
```


### also if yo wanna make it executabal from all directories (after download ofc) use this:
```bash
cd
cd cliTaskManager
sudo cp task /usr/bin
```

### to rebuild file:
```bash
cd
sudo rm /usr/bin/task ~/cliTaskManager/task
cd ~/cliTaskManager/
go build -o task main.go
```
